預設模式：個人單機版

適合學生自我挑戰，免申請帳號。

  • 儲存位置： 資料直接存在學生目前的瀏覽器中。
  • 特性： 學生僅能看到自己的成績與紀錄。
  • 🚨 遺失風險： 若直接在 Gemini 分享網址上玩，重新整理頁面後資料會全數清空。
🚀

升級方案：全班連線版

適合即時競賽、老師遠端掌握全班進度。

  • 優點： 老師端可看即時成績，學生可看全班排行榜。
  • 快速版： 請 AI 「使用 Firebase SDK（免 API）」，僅限 Gemini 分享網址使用。
  • 專業版： 若要放到個人 GitHub，需自行申請 Firebase API 金鑰填入（細節請詢問 AI）。

💾 想要重新整理也不消失？

如果是單機版，建議將生成的程式碼發布到您個人的 GitHub 網址，這樣重新整理後資料依然會留在該台電腦中。

但若是清除快取等瀏覽資料，資料仍會消失。
